Good Visual Objects

"Whirlpool Galaxy"

M51, NGC5194

R.A. -> 13h 29m 52.7s
DEC -> +47° 11' 43"
Months -> March - August

Object View -> Complete
Eyepiece -> 24mm
Moon Glow -> Less than 1/2 Moon

Guest Level -> Hard
Visual Appearence -> Faint, fuzy cores of both galaxies. Very faint disk around large galaxy. Spirals can be faintly visible with good eyes.

"Orion Nebula"

M42, NGC 1976

R.A. -> 05h 35m 17.3s
DEC -> -05° 35' 28"
Months -> November - March

Object View -> Complete
Eyepiece -> 24mm
Moon Glow -> Any

Guest Level -> Easy - Medium
Visual Appearence -> Blue-grey or green-grey, fuzzy, bright cloud in a crescent shape with a patch in the middle. 4-5 bright stars in the center cloud.

"Sombrero Galaxy"

M104, NGC 4594

R.A. -> 13h 42m 11.62s
DEC -> +28° 22' 38.2"
Months -> February-September

Object View -> Complete
Eyepiece -> 41mm
Moon Glow -> Any

Guest Level -> Easy
Visual Appearence -> Bright clump of yellow-white stars. Looks as though you dropped a salt pile onto black construction paper.
