5.5" TEC Telescope

Good Visual Objects

"Pleiades"

M45, "Seven Sisters"

R.A. -> 03h 47m 24s
DEC -> +24 07' 00"
Months -> October-February

Object View -> Complete
Eyepiece -> 30mm 3"
Moon Glow -> Any

Guest Level -> Easy
Visual Appearence -> Bright, blue-white stars (points of light).

"Andromeda Galaxy"

M31, NGC 224

R.A. -> 00h 42m 44.3s
DEC -> +41 16' 09"
Months -> August-December

Object View -> Complete
Eyepiece -> 30mm 3"
Moon Glow -> Less than 1/2 Moon

Guest Level -> Medium-Hard
Visual Appearence -> Faint smudgy oval cloud spans the eyepeice. On no moon nights, M110 and M32 are visible as smaller clouds.
