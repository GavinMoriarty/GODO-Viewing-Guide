Welcome to a fun website, based off of the GOD project.

GOD Project - GODO Object Database

This is a website database of objects that are good visual/photographic targets for the 6 telescopes at the Giovale Open Deck Observatory at Lowell Observatory in Flagstaff, Arizona.


This website is not officially affiliated with Lowell Observatory and in no way represents the views, opinions, or quality of Lowell Observatory and its staff.
