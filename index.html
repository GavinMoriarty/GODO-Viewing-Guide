GODO Object Database Project

This project aims to house a selections of objects that are typically seen through the various telescopes at the Giovale Open Deck Observatory at Lowell Observatory in Flagstaff, AZ.

The purpose is to be a quick reference guide for educators to find an object to view with the telescope they are assigned to. Information shown will be useful in finding the object, setting up the telescope for viewing, and describing what the guest experience would be like.

Simply select on the telescope of choice and scroll through the object list (or search the page) for the object that best suits your need. Keep in mind the time of year, moon glow, and guest experience when selecting the object for the night.
